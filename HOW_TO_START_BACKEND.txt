⚠️ ISSUE: Backend server stops after starting

✅ SOLUTION: Start backend in a terminal and KEEP IT OPEN

═══════════════════════════════════════════════════════════════

🚀 STEP-BY-STEP (3 SIMPLE STEPS):
═══════════════════════════════════════════════════════════════

STEP 1: Open a New Terminal
────────────────────────────────────────────────────────────
In VS Code:
  → Click "Terminal" menu at top
  → Click "New Terminal"
  → A terminal opens at bottom

OR Press: Ctrl + Shift + ` (backtick key)


STEP 2: Start the Backend Server
────────────────────────────────────────────────────────────
Type this command in the terminal:

    cd backend
    npm start

Press Enter and WAIT. You should see:

    🚀 Server running on port 5000
    📍 API available at http://localhost:5000
    ✅ MongoDB Connected Successfully

✅ DO NOT CLOSE THIS TERMINAL!
   Keep it open in the background.


STEP 3: Login to the Application
────────────────────────────────────────────────────────────
The browser is already open at: http://localhost:5173

Click one of these buttons:
  👑 Purple Button  = Admin Login
  🏢 Blue Button    = Company Login
  👤 Green Button   = User Login

Then click "Sign In"

✅ YOU'RE IN!
