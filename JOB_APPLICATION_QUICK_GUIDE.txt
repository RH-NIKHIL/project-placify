👤 FOR USERS (Job Seekers):
═══════════════════════════════════════════════════════════════

HOW TO APPLY FOR A JOB:

1. Login: user@user.com / user123
2. Click: 💼 Job Placement (sidebar)
3. Find a job you like
4. Click: "Apply Now" button
5. Fill the form:
   
   RESUME (Required):
   ✓ Paste Google Drive link
   ✓ OR paste your qualifications:
     
     Example:
     Software Engineer - 3 Years Experience
     Skills: React, Node.js, MongoDB, AWS
     Education: B.S. Computer Science
     Projects: 10+ web applications
     
   COVER LETTER (Optional):
   ✓ Explain why you want the job
   ✓ Highlight relevant experience
   ✓ 200-300 words recommended

6. Click: "Submit Application"
7. ✅ Done! Wait for company response

🏢 FOR COMPANIES:
═══════════════════════════════════════════════════════════════

HOW TO VIEW APPLICATIONS:

METHOD 1:
1. Login: company@company.com / company123
2. Dashboard shows your jobs
3. Each job card shows application count
4. Click: "View Applications →"

METHOD 2:
1. Click: "Registered Students" (top nav)
2. See ALL applications from ALL jobs

WHAT YOU SEE:
✓ Applicant name & email
✓ Applied date & time
✓ Resume/qualifications
✓ Cover letter (if provided)
✓ Current status
✓ Status dropdown to update

HOW TO UPDATE STATUS:
1. Find the application
2. Click status dropdown
3. Select new status:
   - Pending (default)
   - Reviewed (you've seen it)
   - Shortlisted (interested)
   - Rejected (not a fit)
   - Accepted (offer sent)
4. Status updates automatically
